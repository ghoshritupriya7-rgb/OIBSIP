# Data Science Task 4 — Email Spam Detection with Machine Learning

## Oasis Infobyte SIP — Data Science Track

### Objective
Build a Natural Language Processing (NLP) binary classifier that distinguishes spam messages from legitimate (ham) messages using TF-IDF feature extraction and machine learning classifiers.

### Tech Stack
- Python
- pandas
- scikit-learn (TfidfVectorizer, MultinomialNB, LogisticRegression)
- NLTK (stopword removal)
- matplotlib / seaborn
- wordcloud

### Dataset
SMS Spam Collection Dataset (Kaggle / UCI Machine Learning Repository)
- 5,572 messages labeled as `ham` or `spam`
- Loaded from a zipped archive and extracted at runtime
