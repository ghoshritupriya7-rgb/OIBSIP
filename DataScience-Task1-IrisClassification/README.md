# Task 1 - Iris Flower Classification

Trained ML classifiers to identify Iris species (setosa, versicolor, virginica) 
from physical measurements.

## Tech Stack
Python, scikit-learn, pandas, matplotlib, seaborn, Jupyter Notebook

## Approach
- Loaded the Iris dataset from sklearn.datasets
- Performed EDA and visualized feature distributions by species
- Trained Logistic Regression and K-Nearest Neighbors classifiers
- Evaluated using accuracy, confusion matrix, and classification report

## Results
- Logistic Regression: 96.7% accuracy
- KNN (k=5): 100% accuracy — best performing model
