#Task 2 - Unemployment Analysis with Python

Performed exploratory data analysis on India's unemployment data to uncover 
regional and temporal trends, including the impact of COVID-19.

## Tech Stack
Python, pandas, matplotlib, seaborn, Jupyter Notebook

## Approach
- Loaded and cleaned the "Unemployment in India" dataset (stripped whitespace 
  from column names, handled missing values, converted dates)
- Performed EDA: descriptive statistics, region-wise trends
- Time-series analysis of unemployment rate across regions
- Identified top 10 regions by average unemployment rate
- Built a correlation heatmap between unemployment, employment, and labour 
  participation rate
- Compared Pre-COVID vs Post-COVID unemployment rates

## Key Findings
- Tripura and Haryana had the highest average unemployment rates (25%+)
- Unemployment nearly doubled post-COVID (~9% to ~17.5%)
- Clear correlation between unemployment, employment, and labour participation
